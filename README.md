# Exam Prep Tracker

Это статический сайт-трекер, собранный из Excel-файла `Трекер_подготовки_к_экзамену_с_LeetCode.xlsx`.

## Что внутри

- `index.html` — главная страница.
- `styles.css` — оформление.
- `app.js` — данные, логика фильтров, списковое представление тем и сохранение прогресса.
- `.nojekyll` — отключает обработку Jekyll на GitHub Pages.

## Как открыть локально

Самый простой способ: дважды открыть `index.html` в браузере.

Более правильный способ через локальный сервер:

```bash
python -m http.server 8000
```

После этого открой:

```text
http://localhost:8000
```

## Как задеплоить через GitHub Pages

```bash
git init
git add .
git commit -m "feat: add exam prep tracker site"
git branch -M main
git remote add origin https://github.com/<USERNAME>/<REPOSITORY>.git
git push -u origin main
```

Дальше на GitHub:

1. Открой репозиторий.
2. Перейди в `Settings`.
3. Перейди в `Pages`.
4. В разделе `Build and deployment` выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Сохрани настройки.

Сайт будет доступен по адресу:

```text
https://<USERNAME>.github.io/<REPOSITORY>/
```

## Представление тем

Темы показаны в формате списка: они сгруппированы по разделам, а у каждой темы есть компактная строка со статусом, уровнем, неделей, самооценкой и раскрываемым блоком деталей.

## Важно про сохранение прогресса

Прогресс хранится в `localStorage` браузера. Это значит:

- если открыть сайт в другом браузере или на другом устройстве, прогресс не подтянется автоматически;
- для переноса используй кнопки `Экспорт прогресса` и `Импорт прогресса`;
- если очистить данные сайта в браузере, локальный прогресс может удалиться.

## Как обновлять сайт

После изменений в файлах:

```bash
git add .
git commit -m "update tracker"
git push
```

GitHub Pages сам опубликует новую версию после push.

## Обновление дизайна

В этой версии верхний блок заменён на инфографику прогресса: общий процент готовности, прогресс по темам, задачам, конспектам, пирамида уровней A/B/C, прогресс по разделам и трекер на 11 недель.

## Обновление под 11 недель

В этой версии темы и задачи не сокращались: в трекере остаются все 172 темы и 83 задачи практикума. Изменён только срок подготовки: сайт показывает, что до экзамена осталось 11 недель, и рассчитывает примерный темп прохождения от текущего прогресса.

Исходный трекер на 11 недель оставлен как учебная карта, но поверх него добавлен блок сжатого темпа: сколько тем и задач нужно закрывать в неделю, чтобы успеть пройти всю программу.

## Как сайт отслеживает недели

В верхнем блоке есть поле `Дата экзамена`.

Если дату экзамена не задавать, сайт работает в ручном режиме и считает, что осталось 11 недель.

Если дату экзамена сохранить, сайт каждый день автоматически пересчитывает:

- сколько календарных дней осталось до экзамена;
- сколько недель осталось;
- какая сейчас учебная неделя в 11-недельном режиме;
- какие недели уже прошли;
- цель на день: сколько тем, задач и повторения нужно закрыть с учётом уже отмеченного прогресса.

Дата экзамена и весь прогресс хранятся локально в браузере. Для переноса на другое устройство используй `Экспорт прогресса` и `Импорт прогресса`.
